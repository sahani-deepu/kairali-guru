### Interim stock candidates mapped to slots · 2026-07-09

**Read this first.** These are **candidate** free-license images to get the team moving — **not** final choices. Two rules before using any of them:
1. **First-party still wins.** Use these only as interim/supporting fillers; replace with your own healing-village/physician/student photography as soon as you have it (better for premium + E-E-A-T). Faculty, testimonials, and the Delhi/Kerala venues = **first-party only** (no stock).
2. **Verify + download from the source page; don't hotlink.** Pexels & Unsplash = free for commercial use, **no attribution required**; Pixabay = royalty-free — but confirm the license on each page and download the file. **Model-release caveat:** many show recognizable people; stock licenses permit commercial use but not necessarily endorsement-style use in a health context — prefer first-party for any people-forward hero.
3. **Avoid the spa cliché.** A lot of "ayurveda therapy" stock is generic spa (candles, a relaxed Western model) — exactly what §28.2 bans. Filter hard toward authentic Kerala/clinical framing; the Kerala landscape/architecture set below is the safest.
